# Stage 7.4 Multi-backbone Generalization Plan

## 1. Purpose

Stage 7.4 tests whether the proposed anomaly-crop reasoning pipeline depends only on PatchCore or can generalize to other anomaly localization backbones.

The core question is:

Does anomaly-crop reasoning remain effective when candidate regions are generated by another industrial anomaly detector?

## 2. Completed Baseline

PatchCore has already been evaluated on VisA 12 categories.

PatchCore provides:
- image-level anomaly detection results
- pixel-level anomaly maps
- anomaly candidate regions
- full-image versus anomaly-crop VLM reasoning results

## 3. Candidate Backbones

The candidate anomaly backbones are:

1. EfficientAD
2. FastFlow
3. Reverse Distillation
4. STFPM

The first priority is EfficientAD because it is a common teacher-student industrial anomaly detection baseline and is usually faster than large reconstruction-style models.

## 4. Fair Comparison Rule

For every backbone, the comparison must keep the following fixed:

- Dataset: VisA
- Category set: same 12 categories
- Test image set: same 2162 test images
- VLM model: CLIP ViT-B/32
- Prompt strategy: inspection_binary
- Reasoning modes: full_all, crop_or_full, crop_topk_ensemble
- Candidate top-k: 3
- Metrics: AUROC, AP, Best F1, Accuracy
- Candidate coverage: reported separately

## 5. Stage 7.4 Substeps

Stage 7.4-A:
Check which Anomalib model classes are available in the current environment.

Stage 7.4-B:
Run one-category smoke test using the first available non-PatchCore backbone.

Stage 7.4-C:
Generate candidate regions from the new backbone.

Stage 7.4-D:
Run VisA binary prompt reasoning using the new candidate regions.

Stage 7.4-E:
Compare PatchCore versus the new backbone under the same crop reasoning protocol.

## 6. Expected Paper Value

If another backbone also improves crop reasoning over full-image prompting, the paper can claim backbone-level generalization.

If only PatchCore works well, the paper can report a limitation: the method depends on the quality and stability of anomaly localization.
