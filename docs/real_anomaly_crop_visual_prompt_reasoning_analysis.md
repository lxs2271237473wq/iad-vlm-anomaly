Top-k crop ensemble does not outperform top-1 crop in Top-1 accuracy under the current candidate set.\n\n## 5. Conclusion\n\nStage 6.3 is a positive diagnostic result.\n\n```text\nRefined visual prompts become more effective when the input is focused on real anomaly crops.\nThe current bottleneck is candidate crop coverage, not prompt design.\n```\n\n## 6.
