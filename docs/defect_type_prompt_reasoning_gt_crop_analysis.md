# GT-crop Defect Type Prompt Reasoning Analysis\n\n## 1. Purpose\n\nThis document analyzes Stage 6.1-D: GT-mask crop upper-bound defect type reasoning.\n\nThe purpose is to test whether defect type recognition improves when the visual input is focused on the true defect region.\n\nThis is an upper-bound diagnostic experiment because GT masks are used for cropping.\n\n## 2. Best Result\n\n| Metric | Value |\n|---|---|\n| Best Strategy | category_aware |\n| Best Image Mode | gt_crop |\n| Top-1 Accuracy | 0.2973 |\n| Top-2 Accuracy | 0.5022 |\n| Macro-F1 | 0.2147 |\n\n## 4. Observations\n\n1. GT-crop improves defect type recognition over full-image reasoning.\n\n2. The best result is obtained by category-aware prompts with GT crops.\n\n3. Manufacturing-aware prompts are not the strongest classification prompts in the current form.\n\n4. This suggests that long manufacturing-process descriptions introduce semantic noise for CLIP-based classification.\n\n5. Manufacturing knowledge should be used for explanation and cause reasoning, while defect type classification should use shorter, more visual prompts.\n\n## 5. Conclusion\n\nThe GT-crop upper-bound experiment supports the importance of accurate anomaly-region localization.\n\n```text\nBetter anomaly crops improve defect type reasoning.\nCurrent manufacturing-aware prompts are useful for explanation, but not optimal as CLIP classification prompts.\n```\n\n## 6. Next Direction\n\nRecommended next stage:\n\n```text\nStage 6.2: Visual prompt refinement for defect type recognition\n```\n\nThe next prompt design should separate:\n\n- short visual defect prompts for classification\n- manufacturing process knowledge for explanation\n- possible cause templates for report generation\n
